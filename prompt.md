# Intra-sentential Code-Switching Tagging Prompt

This prompt is used for word-level language identification in Ukrainian-Russian mixed speech.

## Prompt:
I'm going to give you a sentence with Ukrainian and Russian words. Please put :uk after each Ukrainian word and :ru after each Russian word.
In format: а:uk живуть:uk за:uk принципом:uk ті:uk оціночники:uk мы:ru не:ru сеем:ru не:ru пашем:ru не:ru строим:ru мы:ru гордимся:ru общественным:ru строем:ru
